## 下载

### Docker

```
docker pull apachecn0/apachecn-cv-zh
docker run -tid -p <port>:80 apachecn0/apachecn-cv-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install apachecn-cv-zh
apachecn-cv-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g apachecn-cv-zh
apachecn-cv-zh <port>
# 访问 http://localhost:{port} 查看文档
```
